# 💎 Nida J. - Legacy of Luxury

A high-end, full-stack AI-powered jewelry e-commerce platform built using **Next.js (App Router)** and the **MERN Stack** architecture. This application blends premium aesthetics with modern AI orchestration, featuring a synchronized real-time digital twin shopping assistant.

---

## ✨ Key Architectural Features

- **Legacy Intelligence (AI Chatbot):** Features an inline production AI assistant designed to act as a digital twin concierge, guiding clients through premium collections and handling checkout inquiries.
- **Intelligent Local Isolation:** Custom localized chat hooks (`useChatStorage`) that isolate historical chat states uniquely per authenticated user, preventing session leakage between client registries.
- **Modern Order Vault Architecture:** A robust checkout pipeline built with structured validation that auto-synches data fields for dynamic order management, returns, and cancellation queues.
- **Premium UX/UI:** Tailored animations, responsive product card layers with elegant hover triggers, multi-category catalogs, and optimized asset delivery using Tailwind CSS.

---

## 🛠️ Tech Stack & Registries

- **Frontend Framework:** Next.js 14/15+ (App Router) & React
- **Styling & Motion:** Tailwind CSS & Framer Motion
- **State Management:** Redux Toolkit (Synchronized Cart & Wishlist States)
- **Database Locker:** MongoDB Atlas via Mongoose ODM
- **AI Orchestration:** Custom-built Digital Twin Assistant via OpenAI API
- **Authentication:** JWT-based secure session management with LocalStorage isolation
- **Security & Optimization:** Bcrypt password hashing, MongoDB Indexing for faster query performance
- **Deployment & Hosting:** Vercel Production Environment
- **Version Control & Management:** GitHub & npm
- **Design Station:** Figma (UI/UX Concept & Prototyping)
